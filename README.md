This is a collection of code for the 250c Regression in Psychology class for incoming graduate students in the UCLA psychology doctoral program.

I am putting the collection here for future-proofing and for anyone to browse if desired. However, some of the material might not make sense outside of the context of the class and labs (e.g., direct references to slide/note material from the course lectures).

The PROCESS materials refer to the PROCESS v5.0 for R macro available from processmacro.org. If one wants to use the macro, download it from the site, load the process.R file in the PROCESS v5.0 for R folder, and run the entire script. This will provide the process() function in the R environment for use. However, all the course material provided covers methods that do not involve this macro.
